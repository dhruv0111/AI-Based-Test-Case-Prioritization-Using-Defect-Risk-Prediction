Analysis and Discussion

The evaluation results indicate that AI-based test case prioritization consistently achieves higher APFD values compared to random and original test execution orders. This improvement demonstrates that incorporating defect risk prediction enables earlier detection of faults during test execution.

The effectiveness of the proposed approach can be attributed to the use of machine learning-derived defect risk scores, which allow testing efforts to be focused on high-risk software modules. By prioritizing test cases associated with these modules, the approach aligns test execution with areas most likely to contain defects.
